# Role Tailoring

## Backend Engineer

Lead with FastAPI, SQLAlchemy models, event ingestion, WebSocket scaffolding, config APIs, service boundaries, internal RBAC, audit logs, incident cases, API keys, and notification escalation.

## ML Engineer

Lead with edge inference, ONNX/TensorRT boundary, risk scoring, occupancy prediction, model card, benchmark script, self-correction loop, labeling queues, and model registry promotion gates.

## Fullstack Engineer

Lead with the React operations console, API integration, fallback data mode, lock controls, incident triage, and live detection visualization.

## DevOps / Cloud Engineer

Lead with Docker Compose, Kubernetes, Terraform topology, GitHub Actions, systemd edge units, health checks, observability config, SLO metrics, dead-letter queues, job runs, and edge command queues.

## Agentic AI Engineer

Lead with incident reviewer, critique generation, human-label gate, retraining proposal queue, and policy recommendation loop.

## Security / Platform Engineer

Lead with tenant isolation, scoped API keys, RBAC decisions, audit logging, incident lifecycle, data retention, device certificates, feature flags, and model governance controls.
